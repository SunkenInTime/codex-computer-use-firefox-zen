Enable the Codex connection

Firefox needs a small local companion to connect this extension to your installed Codex host. It runs only while Codex is connected and does not operate a remote service.

  3. Close this page and reopen the Codex sidebar.

This independent compatibility project is not an official OpenAI or Mozilla release. Installer source and checksums are included with every GitHub release.
